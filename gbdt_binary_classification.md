## Ch 1. 兩階段輸出：為什麼 GBDT 不直接吐機率

直覺上，一個分類模型拿到樣本 $x$，「應該」直接吐出機率 $p(x) \in [0, 1]$。實際上 GBDT 的內部運作不是這樣 ── 它先算一個實數，再把這個實數轉成機率。

### 1.1 兩個輸出階段

對一個樣本 $x$，GBDT 訓練好之後做兩件事：

**Step 1**：算出一個實數 $F(x)$，值可以是任何 $-\infty < F(x) < +\infty$。這個 $F$ 叫做 **raw score**（原始分數），是模型內部產生的東西。

**Step 2**：用 sigmoid 函數把 $F(x)$ 壓到 $[0, 1]$ 區間，得到**正類發生的機率** $p(x)$：

$$p(x) = \sigma\bigl(F(x)\bigr) = \frac{1}{1 + e^{-F(x)}}$$

其中 $\sigma$ 是 sigmoid 函數（→ 等價形式見附錄 A.2）。在二元分類裡，$p(x)$ 是「$x$ 屬於正類（$y = 1$）的機率」，而 $1 - p(x)$ 就是「屬於負類（$y = 0$）的機率」。

本文接下來要回答的核心問題就是 **「$F(x)$ 是怎麼算的？」**

### 1.2 為什麼要繞兩階段，不直接輸出機率？

下一章會看到，GBDT 是「很多棵樹的輸出加總」。每棵樹輸出一個實數（可正可負）。把這些實數直接相加完全沒問題，加完還是實數。

但如果樹輸出的是「機率調整量」，加起來會發生兩件糟糕的事：

1. **超界**：機率必須在 $[0, 1]$ 之間。$0.7 + 0.4 = 1.1$ 在這個世界沒有意義。
2. **不平等**：把 $p$ 從 $0.50$ 推到 $0.60$，跟從 $0.95$ 推到 $1.00$，看起來都是 +0.10，但實際上後者要難得多。在 log-odds 空間裡，從 $p = 0.5$（$F = 0$）到 $p = 0.6$（$F \approx 0.405$）只需要 $F$ 移動約 $0.4$；但從 $p = 0.95$（$F \approx 2.944$）推到 $p = 1$，$F$ 必須跑到 $+\infty$。「機率越接近邊緣，每微小變動越貴」這件事，加法在 log-odds 空間會自動編碼進去 ── 因為 $F = \log\frac{p}{1-p}$ 把 $[0, 1]$ 拉成 $(-\infty, +\infty)$，$p$ 越靠邊 $F$ 走得越遠；所以「$F$ 空間加固定量」對應到「$p$ 空間靠邊時微動、中間時大動」這種自然不均勻變化。加在機率空間則一視同仁，不合理。

GBDT 的設計是：**讓樹在實數空間 $F$ 裡做加法，最後一次 sigmoid 把實數壓回機率**。Step 1 的累加不受機率界限拘束；Step 2 才負責讓輸出長得像機率。

### 1.3 $F$ 的另一個名字：log-odds

從 $p = \sigma(F)$ 反過來解 $F$，可以得到一個有意義的結果。

先把 sigmoid 展開：

$$p = \frac{1}{1 + e^{-F}}$$

兩邊乘 $(1 + e^{-F})$：

$$p\,(1 + e^{-F}) = 1$$

展開：

$$p + p\,e^{-F} = 1$$

移項：

$$p\,e^{-F} = 1 - p$$

兩邊除以 $p$：

$$e^{-F} = \frac{1 - p}{p}$$

兩邊取自然對數：

$$-F = \log\frac{1 - p}{p}$$

翻負號（利用 $-\log(a/b) = \log(b/a)$）：

$$\boxed{\;F = \log\frac{p}{1 - p}\;}$$

所以 $F$ 本身有具體意義：它是 **「正類 vs 負類 odds 的對數」**。

- $p/(1-p)$ 叫做 odds（勝率）：例如 $p = 0.75$，odds $= 0.75/0.25 = 3$，「正類發生的可能性是負類的 3 倍」
- 對 odds 取 $\log$ 後，就是 $F$
- 統計學把這個量叫做 **log-odds** 或 **logit**

GBDT 的 raw score $F$ 和 logistic regression 的線性組合 $w^T x + b$ 屬於同一個量 ── 都是 log-odds。兩者的最後一步都一樣：透過 sigmoid 把 $F$ 轉成機率 $p = \sigma(F)$。差別只在「這個 $F$ 怎麼算出來」── logistic regression 用特徵的線性組合算，GBDT 用一堆樹的加總算。

### 1.4 從 $F$ 到「預測類別」

模型若要做硬決策（不只給機率，要給「是 / 否」），標準做法是 $p \ge 0.5$ 就預測正類：

$$\hat y = \mathbb{1}\bigl[p(x) \ge 0.5\bigr]$$

其中 $\mathbb{1}[\cdot]$ 是**指示函數**（indicator function）：方括號內條件為真時等於 $1$，否則等於 $0$。

因為 sigmoid 在 $F = 0$ 處剛好等於 $0.5$（驗算：$\sigma(0) = 1/(1+e^0) = 1/2$），上面的條件等價於：

$$\hat y = \mathbb{1}\bigl[F(x) \ge 0\bigr]$$

換句話說，**門檻決策實際上發生在 $F$ 的空間**：正類 vs 負類的分界線是 $F = 0$。$F$ 越正越偏正類、越負越偏負類。模型內部一直在跟 $F$ 打交道，只是最後給人看的時候轉成機率而已。

### 1.5 範例：sigmoid 數值對照

幾個典型 $F$ 值對應的 $p$：

| $F$ | $p = \sigma(F)$ | 解讀 |
|---|---|---|
| $-3$ | $0.047$ | 強烈偏負類 |
| $-1$ | $0.269$ | 偏負類 |
| $\phantom{-}0$ | $0.500$ | 中立（門檻）|
| $+1$ | $0.731$ | 偏正類 |
| $+3$ | $0.953$ | 強烈偏正類 |

注意 $F = 0$ 剛好對應 $p = 0.5$（1.4 的等價條件），這就是硬決策的分界。

sigmoid 是 $S$ 形：靠近 $F = 0$ 對 $p$ 影響最劇烈（$F$ 從 $0$ 到 $1$ 推 $p$ 從 $0.5$ 到 $0.73$，移動 $0.23$），離 $0$ 越遠對 $p$ 影響越弱（$F$ 從 $1$ 到 $3$ 才把 $p$ 從 $0.73$ 推到 $0.95$，只移動 $0.22$ 卻跨了兩單位）。換句話說：$p$ 在邊界（接近 $0$ 或 $1$）時，需要在 $F$ 空間走很遠才能改變一點。

## Ch 2. $F(x)$ 的加法結構

$F(x)$ 怎麼算？GBDT 的回答是：**$F(x)$ 是一堆樹的輸出加起來**。具體形式：

$$F(x) = F_0 + \nu \sum_{m=1}^{M} f_m(x)$$

### 2.1 每個符號是什麼

| 符號 | 意義 |
|---|---|
| $M$ | 樹的總棵數（超參數，例如 100、500、1000） |
| $f_m(x)$ | 第 $m$ 棵樹對樣本 $x$ 的輸出（實數，可正可負） |
| $F_0$ | 起始常數，跟 $x$ 無關，所有樣本共用一個值（Ch 3 推導怎麼算） |
| $\nu$ | learning rate（學習率），縮放每棵樹的貢獻，通常 $0.05 \sim 0.3$，預設常見 $0.1$ |

訓練好的模型由「$F_0$ 一個數字 + $M$ 棵訓練好的樹」組成；預測時把 $x$ 餵進去，每棵樹各吐一個數字，全部（有 $\nu$ 縮放）加上 $F_0$ 就是 $F(x)$。

### 2.2 為什麼要加 learning rate $\nu$

每棵樹是「給定當下模型狀態」下找到的最佳修正。具體機制 Ch 4–6 會推導；簡單講：下一棵樹的訓練目標是「讓當下預測變得更好」，所以它會自然朝著「需要修正」的方向走。但「當下狀態下最佳」是相對的 ── 一旦這棵樹加進去，模型狀態就變了；下一棵樹會根據新狀態重新找方向。

如果第一棵樹毫不打折地直接加，整個 $F$ 就被一口氣推得很遠；第二棵樹發現「上一棵走過頭了」，反方向去補；補完又被第三棵推遠 ── 訓練過程會在來回震盪中失穩。

學習率 $\nu < 1$ 把每棵樹的貢獻乘以 $\nu$（例如 $\nu = 0.1$ 就是只取 $1/10$ 的修正量）。這帶來兩個效果：

- **訓練變慢**：同樣的目標需要更多棵樹才能收斂（每棵只走 $\nu$ 步）
- **泛化變好**：每棵樹的貢獻被縮小，最終預測由多棵樹共同累積，不依賴少數幾棵的判斷；單棵樹過擬合訓練集噪音的影響被稀釋

這是 boosting 領域熟知的 **shrinkage**：小步多走比大步少走好。實務上 $\nu$ 和 $M$（樹的數量）會一起調 ── $\nu$ 越小、$M$ 要越大。

### 2.3 為什麼是「加」？對應到 odds 空間是「乘」

加法形式（在 $F$ 空間）計算很方便，但 $F$ 是個抽象的實數。把它轉回 odds 空間，可以看見一個更直覺的等價形式。為什麼想看 odds？因為 odds $= p / (1 - p)$ 有具體意義 ──「正類發生機率是負類的幾倍」── 比 log-odds 好讀。

設 $\text{odds}_0 = p_0 / (1 - p_0)$ 是 $F_0$ 對應的起始 odds。把 $F(x) = \log(\text{odds}(x))$（Ch 1.3 的結果）代回加法模型：

$$\log\bigl(\text{odds}(x)\bigr) = \log(\text{odds}_0) + \nu f_1(x) + \nu f_2(x) + \cdots + \nu f_M(x)$$

兩邊取 $\exp$，利用恆等式 $e^{a + b} = e^a \cdot e^b$（**log 的加法 $\leftrightarrow$ exp 的乘法**）：

$$\text{odds}(x) = \text{odds}_0 \cdot \prod_{m=1}^{M} e^{\nu f_m(x)}$$

也就是：**$F$ 空間的加法 $=$ odds 空間的乘法**。每棵樹貢獻一個 **odds 倍率** $e^{\nu f_m(x)}$，全部 $M$ 個倍率乘到 $\text{odds}_0$ 上（乘法可交換，所以順序不影響結果）。

每棵樹輸出的 $f_m(x)$ 是第 $m$ 棵樹對 $x$ 的輸出，**以 log-odds 為單位**（後面 Ch 4–6 會看到怎麼算出來）。它的正負號和大小有具體解讀：

- $f_m(x) > 0$：倍率 $e^{\nu f_m(x)} > 1$，odds 被往上推（更偏正類）
- $f_m(x) < 0$：倍率 $e^{\nu f_m(x)} < 1$，odds 被往下推（更偏負類）
- $|f_m(x)|$ 越大：這棵樹對 $x$ 的判斷越「強」

所有樹的判斷以乘法累積成最終 odds，最後再 sigmoid 變成 $p(x)$。

### 2.4 範例：log-odds 加法 vs odds 乘法 雙演算

假設某樣本起點 $F_0 = 0$（對應 $\bar y = 0.5$ 的資料），模型蓋了 2 棵樹，對該樣本的輸出是 $f_1(x) = +0.5$、$f_2(x) = +1.0$。為了驗算方便取 $\nu = 1$。

**log-odds 空間加法**（直接套 $F = F_0 + \nu \sum f_m$）：

$$F_2(x) = 0 + 0.5 + 1.0 = 1.5, \qquad p(x) = \sigma(1.5) \approx 0.818$$

**odds 空間乘法**（套 odds 連乘版本）：

$$\text{odds}_0 = \frac{0.5}{0.5} = 1, \quad e^{0.5} \approx 1.649, \quad e^{1.0} \approx 2.718$$

$$\text{odds}(x) = 1 \times 1.649 \times 2.718 \approx 4.482$$

$$p(x) = \frac{4.482}{1 + 4.482} \approx 0.818 \;\checkmark$$

兩條路徑算出相同 $p(x) \approx 0.818$，確認「$F$ 空間加法 $\Leftrightarrow$ odds 空間乘法」。直覺解讀：兩棵樹各推 odds 一個倍率（$1.649$、$2.718$），原本 1:1 的 odds 被推到約 $4.5:1$（正類比負類高出 4.5 倍）。

## Ch 3. $F_0$ 從哪來

$F_0$ 是 $F$ 加法分解裡那個跟 $x$ 無關的常數。怎麼決定它的值？

### 3.1 還沒有任何樹的時候，模型該給什麼值？

第一棵樹還沒蓋之前，模型對任何樣本 $x$ 給出的 $F$ 都只能是一個常數 ── 既然還沒看過任何特徵，所有 $x$ 自然一視同仁。

把這個常數設多少？要找「最有用的初始值」：在訓練資料上**整體 loss 最小**的那個常數。loss 最小意味著「在還沒辦法區分樣本的情況下，這個值對所有樣本的犯錯總和最小」。

### 3.2 寫出問題：最小化整體 loss

logistic loss 對單筆樣本 $(x_i, y_i)$ 的形式是：

$$L(y_i, F) = \log(1 + e^F) - y_i \cdot F$$

它跟二元分類常見的 cross-entropy 形式 $L = -y_i \log p_i - (1 - y_i)\log(1 - p_i)$ 是同一個 loss。把 $p_i = \sigma(F)$ 代入兩個情況分開驗：

- $y_i = 1$ 時：$L = -\log\sigma(F)$。用 $\sigma(F) = \frac{e^F}{1+e^F}$，$L = -\log\frac{e^F}{1+e^F} = -F + \log(1+e^F) = \log(1+e^F) - 1 \cdot F$ ✓
- $y_i = 0$ 時：$L = -\log\bigl(1 - \sigma(F)\bigr)$。用 $1 - \sigma(F) = \frac{1}{1+e^F}$（兩種等價形式相減即得，見附錄 A.2），$L = -\log\frac{1}{1+e^F} = \log(1+e^F) = \log(1+e^F) - 0 \cdot F$ ✓

兩個情況合併寫，就是 $\log(1 + e^F) - y_i F$。

把 $F$ 對所有樣本設成同一個常數 $\gamma$，整體 loss 就是（注意 $\gamma$ 對所有樣本相同，但 $y_i$ 因樣本而異）：

$$\sum_{i=1}^{n} L(y_i, \gamma) = \sum_{i=1}^{n} \bigl[\log(1 + e^\gamma) - y_i \gamma\bigr]$$

我們要找的 $F_0$ 就是讓這個和最小的 $\gamma$，正式寫法：

$$F_0 = \arg\min_{\gamma}\; \sum_{i=1}^{n} \bigl[\log(1 + e^\gamma) - y_i \gamma\bigr]$$

（$\arg\min_\gamma$ 念作「使後面式子最小的那個 $\gamma$」。）

### 3.3 解一階條件

logistic loss 是凸函數 ── 它對 $\gamma$ 的二階導數是 $\frac{d\sigma(\gamma)}{d\gamma} = \sigma(\gamma)(1 - \sigma(\gamma))$（→ 見附錄 A.2 sigmoid 微分），而 $\sigma \in (0, 1)$ 保證 $\sigma(1-\sigma) > 0$ 恆成立。二階導數恆正即凸函數。凸函數的最小值處導數 $= 0$（→ 見附錄 A.1），所以解 $\frac{d}{d\gamma}\sum_i L = 0$ 就找到 $F_0$。

對 $\gamma$ 微分整體 loss（$\sum$ 與微分可交換）：

$$\frac{d}{d\gamma}\sum_{i=1}^{n} \bigl[\log(1 + e^\gamma) - y_i \gamma\bigr]
= \sum_{i=1}^{n} \frac{d}{d\gamma}\bigl[\log(1 + e^\gamma) - y_i \gamma\bigr]$$

逐項微分：

- $\dfrac{d}{d\gamma}\log(1 + e^\gamma) = \sigma(\gamma)$（→ 見附錄 A.2）
- $\dfrac{d}{d\gamma}\bigl(-y_i \gamma\bigr) = -y_i$（$y_i$ 是資料，跟 $\gamma$ 無關，當常數處理）

合起來：

$$\frac{d}{d\gamma}\sum_{i=1}^{n} L(y_i, \gamma)
= \sum_{i=1}^{n} \bigl[\sigma(\gamma) - y_i\bigr]
= n\,\sigma(\gamma) - \sum_{i=1}^{n} y_i$$

（$\sigma(\gamma)$ 對每個樣本都是同一個數值 ── $\gamma$ 是個常數，$\sigma$ 跟 $i$、$y_i$ 都無關，所以「加 $n$ 次同一個數」就是 $n$ 倍那個數：$\sum_{i=1}^n \sigma(\gamma) = n \cdot \sigma(\gamma)$。）

令導數 $= 0$：

$$n\,\sigma(\gamma) - \sum_{i=1}^{n} y_i = 0$$

兩邊除以 $n$：

$$\sigma(\gamma) = \frac{1}{n}\sum_{i=1}^{n} y_i$$

右邊有具體名稱：因為 $y_i \in \{0, 1\}$，$\frac{1}{n}\sum y_i$ 就是「全部樣本中 $y = 1$ 的比例」。給它一個簡稱 $\bar y$（正類比例）：

$$\sigma(\gamma) = \bar y$$

### 3.4 解出 $F_0$

從 $\sigma(\gamma) = \bar y$ 反解 $\gamma$。回憶 Ch 1.3 推過：$p = \sigma(F) \iff F = \log\dfrac{p}{1-p}$。把那裡的 $p$ 換成 $\bar y$、$F$ 換成 $\gamma$，得：

$$\gamma = \log \dfrac{\bar y}{1 - \bar y}$$

這個 $\gamma$ 就是 3.2 定義的 $F_0$（讓整體 loss 最小的那個常數）。所以：

$$\boxed{\;F_0 = \log \dfrac{\bar y}{1 - \bar y}\;}$$

意義：**$F_0$ 是「訓練資料整體正類比例」的 log-odds**。

直覺：模型還沒看任何特徵時，最合理的初始猜測就是「全體正類比例」── 因為前面的推導告訴我們，**這個值剛好就是使訓練集 loss 最小的常數**。例如訓練資料 30% 是正類，那對任何尚未檢視的樣本，先猜「他是正類的機率 $= 30\%$」就是「在無法區分樣本的限制下，整體犯錯總和最少」的選擇。$F_0$ 只是把這個 30% 用 log-odds 表示出來而已。

對所有樣本，第 0 步（還沒蓋樹）的預測機率：

$$p_0 = \sigma(F_0) = \bar y$$

也就是：**還沒有任何樹的時候，每個樣本的 $p$ 都等於 $\bar y$**。等樹開始加進來，$F(x)$ 變成 $F_0 + \nu f_1(x) + \nu f_2(x) + \cdots$，不同樣本走過的樹路徑不同、$f_m(x)$ 值不同，$F(x)$ 與對應的 $p(x)$ 才會逐漸從共同起點分化。

### 3.5 範例：$F_0$ 的具體數值

假設訓練資料 30 筆樣本、其中 10 筆 $y = 1$，所以 $\bar y = 10/30 = 1/3$：

$$F_0 = \log \dfrac{1/3}{2/3} = \log \dfrac{1}{2} = -\log 2 \approx -0.693$$

驗證 $p_0 = \sigma(F_0) = \bar y$：

$$\sigma(-0.693) = \dfrac{1}{1 + e^{0.693}} = \dfrac{1}{1 + 2} = \dfrac{1}{3} \;\checkmark$$

不同正類比例對應的 $F_0$ 對照：

| $\bar y$ | $F_0 = \log\frac{\bar y}{1-\bar y}$ |
|---|---|
| $50\%$ | $\phantom{-}0$ |
| $30\%$ | $-0.847$ |
| $10\%$ | $-2.197$ |
| $\phantom{0}1\%$ | $-4.595$ |

$\bar y$ 越小，$F_0$ 越往負方向走。

## Ch 4. 第 $m$ 棵樹要學什麼

有了 $F_0$ 當起點，第 1 棵樹開始，每棵新樹要解決同一個子問題：**手上現有的模型 $F_{m-1}$，下一棵樹 $f_m$ 該怎麼蓋，才能讓整體 loss 下降？**

「樹該怎麼蓋」分兩件事：樹的結構（哪裡切）跟葉節點的輸出（每葉吐多少）。Ch 5、Ch 6 各自處理。本章先回答更基本的問題：**樹該學的訓練目標（target）是什麼？** 答案會推出一個叫 **pseudo-residual** 的量，全章圍繞它展開。

### 4.1 為什麼要對 loss 微分

寫出第 $m$ 輪「加上 $f_m$ 之後」的整體 loss：

$$\sum_{i=1}^{n} L\bigl(y_i,\, F_{m-1}(x_i) + f_m(x_i)\bigr)$$

我們要找的 $f_m$ 是「能讓這個和變小」的函數。但 $L$ 對 $F$ 是非線性的（logistic loss 形式 $\log(1+e^F) - yF$ 不好直接最佳化），所以用一個經典手法：**在 $F_{m-1}$ 附近做一階線性近似**。

對每筆樣本，把 $L$ 看成 $F$ 的函數，在 $F = F_{m-1}(x_i)$ 處用一階泰勒展開（→ 見附錄 A.3；那裡有完整二階形式 $L(F_0 + \delta) \approx L(F_0) + L'(F_0)\delta + \tfrac{1}{2}L''(F_0)\delta^2$，這裡只取零階 + 一階項，把二階以上丟掉）：

$$L\bigl(y_i,\, F_{m-1}(x_i) + f_m(x_i)\bigr) \;\approx\; L\bigl(y_i, F_{m-1}(x_i)\bigr) + \underbrace{\dfrac{\partial L}{\partial F}\bigg|_{F = F_{m-1}(x_i)}}_{\text{記作 } g_i} \cdot f_m(x_i)$$

也就是說，每筆樣本的「加新樹後的 loss」近似等於「不加新樹的 loss」加上「梯度 $g_i$ 乘以新樹輸出 $f_m(x_i)$」。

對全部樣本加總：

$$\sum_{i=1}^{n} L\bigl(y_i, F_{m-1}(x_i) + f_m(x_i)\bigr)
\;\approx\; \underbrace{\sum_i L\bigl(y_i, F_{m-1}(x_i)\bigr)}_{\text{跟 } f_m \text{ 無關，視為常數}} + \sum_i g_i \cdot f_m(x_i)$$

要讓整體 loss 變小，就要讓 $\sum_i g_i \cdot f_m(x_i)$ 越負越好。**最自然的選擇是讓 $f_m(x_i) = -g_i$**（每筆樣本反向追梯度），這樣每一項都變成 $-g_i^2 \le 0$、總和必為負。

理論上把 $f_m(x_i)$ 取成 $-2g_i$、$-10g_i$ 會讓 $\sum_i g_i f_m(x_i)$ 更負、loss 看起來減得更多 ── 但這是一階近似的「假象」。一階泰勒只在 $\delta = f_m(x_i)$ 小時才精確；$f_m$ 一旦放太大，前面丟掉的二階及更高項會反撲，真實 loss 反而上升。所以 GBDT 的策略是：**先把方向定為 $-g_i$**，再透過「樹結構 + 葉節點輸出 + learning rate $\nu$」這三層控制實際推進的幅度（後三者分別由 Ch 5、Ch 6、Ch 2 已說明的 $\nu$ 處理）。

但樹做不到「對每筆樣本獨立指定輸出」。樹的限制是：所有落在同一片葉子的樣本只能拿到同一個輸出值。所以真正的問題變成 ──「**訓練一棵迴歸樹，讓它對 $x_i$ 的輸出盡量逼近 $-g_i$**」。樹彈性夠大（可以對輸入空間任意分區、每區給一個輸出值），就算每筆逼不到完美，整體仍能做出顯著的 loss 下降。

這個 $-g_i$ ── 也就是「loss 對 $F$ 的負梯度，在當前點 $F_{m-1}(x_i)$ 求值」── 就是樹的訓練目標。下一節算它對 logistic loss 長什麼樣。

順帶一提，這個思路在 boosting 文獻叫 **functional gradient descent**（函數空間的梯度下降）。一般 gradient descent 更新「參數」$\theta$；GBDT 沒有固定參數集合，改成每輪「加一個函數 $f_m$」── 這個函數在每個樣本上的輸出要朝著「讓 loss 變小的方向」走，也就是負梯度方向。

附帶說明 ── 一階 vs 二階的分工：本章只用一階泰勒，就足以決定「樹該往什麼方向學」（target = $-g_i$）。後面 Ch 6 要決定「每片葉子最終輸出多少」時，會升級到二階泰勒，得到更精準的 Newton step 解。

### 4.2 對 logistic loss 微分得到 $r = y - p$

代 Ch 3 寫過的單筆 logistic loss：

$$L(y_i, F) = \log(1 + e^F) - y_i \cdot F$$

對 $F$ 微分。$\log(1 + e^F)$ 的導數是 $\sigma(F)$（→ 見附錄 A.2）；$y_i \cdot F$ 對 $F$ 微分得 $y_i$（$y_i$ 是資料給的常數，不依賴 $F$）。合起來：

$$\frac{\partial L}{\partial F} = \sigma(F) - y_i$$

在第 $m$ 輪的當前點 $F = F_{m-1}(x_i)$ 求值：

$$g_i = \dfrac{\partial L}{\partial F}\bigg|_{F = F_{m-1}(x_i)} = \sigma\bigl(F_{m-1}(x_i)\bigr) - y_i$$

定義 **當前預測機率**：

$$p_{i, m-1} \;=\; \sigma\bigl(F_{m-1}(x_i)\bigr)$$

它代表「進入第 $m$ 輪之前、累積到第 $m-1$ 棵樹結束時，模型對 $x_i$ 給的預測機率」。下標 $m-1$ 強調這是「上一輪結束時的 $F$」推出來的。

代入：

$$g_i = p_{i, m-1} - y_i$$

樹的訓練目標 $-g_i$ 就是：

$$-g_i = y_i - p_{i, m-1}$$

GBDT 文獻給這個量一個專屬名稱 **pseudo-residual**（虛擬殘差），記作：

$$\boxed{\;r_{im} \;=\; y_i - p_{i, m-1}\;}$$

兩個下標：$m$ 表示「第 $m$ 輪的計算結果」、$i$ 表示「樣本編號」。

### 4.3 $r$ 的意義

$r_{im}$ 是「真實標籤」與「進入第 $m$ 輪前模型給的預測機率」的差。分兩種樣本看：

- **正樣本（$y_i = 1$）**：$r_{im} = 1 - p_{i, m-1}$，永遠為正
  - $p$ 越接近 $1$：$r$ 越小（接近 0）── 模型已經對這筆樣本接近確信，殘差小，不太需要修正
  - $p$ 越接近 $0$：$r$ 越大（接近 1）── 模型嚴重低估這筆正樣本，殘差大，急需修正
- **負樣本（$y_i = 0$）**：$r_{im} = -p_{i, m-1}$，永遠為非正
  - $p$ 越接近 $0$：$r$ 越接近 $0$ ── 模型接近確信這是負樣本，殘差小
  - $p$ 越接近 $1$：$r$ 越接近 $-1$ ── 模型嚴重誤判為正樣本，殘差負得越深、急需反向修正

換句話說：**$r$ 自動編碼了「目前模型對每筆樣本的犯錯程度與方向」**。$|r|$ 大代表錯得多、要重點修；$|r|$ 小代表錯得少、不太需要動。樹在「逼近 $r$」這個任務上，同時兼顧了「該往哪個方向修」（$r$ 的正負號）和「該修多少」（$|r|$ 的大小）。

樹的輸出再被加進 $F$ 完成這個閉環：
- 對「真實為 $1$ 但 $p$ 偏低」的樣本（$r > 0$），樹會吐正值 → $F$ 被推大 → 下一輪 $p$ 被推大 → 殘差變小
- 對「真實為 $0$ 但 $p$ 偏高」的樣本（$r < 0$），樹會吐負值 → $F$ 被推小 → 下一輪 $p$ 被推小 → 殘差絕對值變小

第 $m$ 棵樹的訓練資料就是：

$$\bigl\{\bigl(x_i,\, r_{im}\bigr)\bigr\}_{i=1}^{n}$$

用 $x_i$ 當特徵、$r_{im}$ 當迴歸目標。注意這是**迴歸樹**（不是分類樹）：因為 $r_{im}$ 是實數（範圍 $[-1, 1]$，但邏輯上可看作任意實數），需要的是「樹的葉節點吐實數」這種輸出形式 ── 迴歸樹會把樣本分群，每群吐一個能讓「葉內 $r$ 逼近誤差最小」的常數（最簡形式就是葉內 $r$ 的平均；Ch 5、Ch 6 會看到 GBDT 用更精細的策略）。

「樹的結構怎麼長」（哪裡切、切幾刀）跟「葉節點輸出多少」這兩個剩下的問題，分別由 Ch 5、Ch 6 處理。

### 4.4 範例：第一輪所有樣本的 $r$

假設 4 筆樣本（2 正 2 負），這是第 1 棵樹開始之前的狀態，所以每個樣本的當前預測機率都是 $F_0$ 對應的 $p_0 = 0.3$：

| ID | $y_i$ | $p_{i,0}$ | $r_{i,1} = y_i - p_{i,0}$ |
|---|---|---|---|
| A | $1$ | $0.3$ | $+0.7$ |
| B | $1$ | $0.3$ | $+0.7$ |
| C | $0$ | $0.3$ | $-0.3$ |
| D | $0$ | $0.3$ | $-0.3$ |

第 1 棵樹的訓練資料就是 $\{(x_A, +0.7), (x_B, +0.7), (x_C, -0.3), (x_D, -0.3)\}$ ── 樹要學的不是 $y$，是「真實標籤跟當前預測的差」。

正樣本 $|r| = 0.7$ 比負樣本 $|r| = 0.3$ 大 ── 因為 $p_0 = 0.3$（對應 $\bar y = 0.3$），正樣本距離目標 $y = 1$（差 $0.7$）比負樣本距離 $y = 0$（差 $0.3$）遠，所以樹的注意力自動偏向正樣本。

## Ch 5. 樹的結構：一棵樹怎麼長出來

樹的訓練目標已知（$r = y - p$，Ch 4 推得），但「樹要學 $r$」只是訂方向，沒說怎麼蓋。本章專注於一件事：**給定訓練資料 $\{(x_i, r_i)\}$，要怎麼決定樹的結構**（哪裡切、切幾刀）。

### 5.1 一棵樹的解剖

一棵決策樹是一個函數 $f: \mathbb{R}^d \to \mathbb{R}$（$d$ 是特徵維度）。內部結構：

樹由「節點」組成，分兩種：

- **內部節點（internal node）**：每個內部節點掛著一個「分裂規則」── 例如「特徵 $j$ 是否 $\ge$ 閾值 $t$」。樣本流到這個節點時，根據規則往左子節點或右子節點走
- **葉節點（leaf node）**：樹的盡頭，不再分裂。每個葉節點存一個常數 $\gamma_j$，當樣本最後停在這片葉子，這個 $\gamma_j$ 就是樹對該樣本的輸出

樹從一個 **根節點** 開始（所有訓練樣本都在這裡），往下層層分裂，直到每條路徑都走到某個葉節點。

形式化寫法：訓練好的樹 $f$ 把輸入空間 $\mathbb{R}^d$ 切成 $J$ 個互不相交的區域 $R_1, R_2, \ldots, R_J$（每個葉節點對應一個區域），對應 $J$ 個葉節點輸出 $\gamma_1, \ldots, \gamma_J$。樹對 $x$ 的輸出可寫成：

$$f(x) = \sum_{j=1}^{J} \gamma_j \cdot \mathbb{1}\bigl[x \in R_j\bigr]$$

意思：「$x$ 落在哪一片葉子（哪個區域 $R_j$），$f(x)$ 就是那片葉子的常數 $\gamma_j$」。因為每個 $x$ 只能落在一個 $R_j$ 裡（區域互不相交），這個 $\sum$ 裡每次只有一項是 $1$，其他都是 $0$。

一棵迴歸樹要訓練的就是兩件事：
- **結構**：怎麼切？每個內部節點用哪個特徵、哪個閾值？這也決定了區域 $R_1, \ldots, R_J$ 的形狀
- **葉節點輸出**：每片葉子的 $\gamma_j$ 是多少？

本章處理結構。葉節點輸出 Ch 6 處理。

### 5.2 樹是怎麼長出來的：greedy 流程

「找出最好的樹」這個問題嚴格來說 NP-hard：可能的樹結構數量隨樣本數、特徵數、深度指數爆炸，沒辦法窮舉所有可能挑最好的。實務上 GBDT（包含 XGBoost、LightGBM）都用 **greedy** 流程逐步生長：

1. **起點**：所有訓練樣本都在根節點
2. **找最佳切分**：對當前節點，掃過所有「特徵 + 閾值」的候選組合，挑出讓「切分後 loss 改善最多」的那一刀（量化指標叫 **Gain**，5.3 給出公式）
3. **切**：根據選中的特徵 + 閾值，把當前節點的樣本分到左、右兩個子節點
4. **遞迴**：對每個子節點重複步驟 2、3
5. **停**：滿足停止條件就停。停下來的節點就成為葉節點，不再分裂。常見的停止條件與動機：
   - 葉節點樣本數過少（太少樣本算出來的方向不可靠，容易過擬合）
   - 深度過深（深度大會讓樹太複雜，泛化變差、計算變慢）
   - 能找到的最佳 Gain 太小（再切下去 loss 也降不了多少，不如停）
   - 葉數達到上限（全樹的容量上限，避免單棵樹過於複雜）

「greedy」的意思是：每一步**只看「當下這一刀讓 loss 減多少」**，不回頭考慮「如果第一刀切別處，後面可不可以更好」。所以這個流程**不保證找到全局最佳的樹**，但實務上效果夠好，計算成本也可控。

### 5.3 量化「切的好壞」：Gain 公式

5.2 第 2 步說「挑讓 loss 改善最多的那一刀」。具體用什麼量化「改善」？答案是 **Gain**。

設當前節點包含一群樣本，每筆樣本帶有兩個梯度資訊：

- $g_i = \dfrac{\partial L}{\partial F}\bigg|_{F = F_{m-1}(x_i)} = p_i - y_i$（**一階導數**，Ch 4.2 已推）
- $h_i = \dfrac{\partial^2 L}{\partial F^2}\bigg|_{F = F_{m-1}(x_i)} = p_i(1 - p_i)$（**二階導數**；嚴格推導見 Ch 6，這裡先採信公式形式）

直觀對照：$g_i$ 描述「loss 在當前點對 $F$ 的斜率」（往哪個方向 loss 下降）；$h_i$ 描述「loss 在當前點的曲率」（曲率越大代表 loss 函數在這附近越「擠」，需要的步長越小才不會跨過最低點）。後面 Gain 公式裡會看到 $H = \sum h_i$ 出現在分母，扮演「步長調節器」的角色。

葉內統計量（節點層級的一階、二階總和）：

$$G = \sum_{i \in \text{node}} g_i, \qquad H = \sum_{i \in \text{node}} h_i$$

切一刀後分成左右兩群，各自的統計量：

$$G_L = \sum_{i \in \text{left}} g_i, \quad H_L = \sum_{i \in \text{left}} h_i, \quad G_R = \sum_{i \in \text{right}} g_i, \quad H_R = \sum_{i \in \text{right}} h_i$$

注意切之前的 $G, H$ 跟切之後的左右兩群有守恆關係（樣本沒消失，只是被分組）：

$$G = G_L + G_R, \qquad H = H_L + H_R$$

在給出 Gain 公式前，先回答一個動機：**為什麼要量化「切的效果」必須用到二階導數 $H$？**

只看一階梯度 $G$ 只能告訴你「往哪個方向修能讓 loss 下降」（梯度方向）。但要量化「切了之後葉節點挑最佳 $\gamma$ 能讓 loss 減多少」（loss 減少量的具體數值），就必須知道 loss 函數在當前點的**曲率**（二階導數）── 曲率告訴我們「最佳步長有多大、對應的 loss 下降值有多少」。這就是為什麼 Gain 公式裡會出現 $H$ ── 它編碼了 loss 的曲率，跟 $G$ 一起決定「葉節點最終能優化到多少 loss 改善」。

**Gain 公式**（嚴格推導見 Ch 6.4，這裡先給結果並建立直覺）：

$$\boxed{\;\text{Gain} = \dfrac{1}{2}\left[\dfrac{G_L^2}{H_L + \lambda} + \dfrac{G_R^2}{H_R + \lambda} - \dfrac{G^2}{H + \lambda}\right]\;}$$

其中 $\lambda \ge 0$ 是 L2 正則化參數（防止葉節點輸出過度爆衝，Ch 6 推導時一併引入）。$\lambda = 0$ 時就是無正則版本。

**直覺**：Gain 公式整個是「二階泰勒近似」應用到「切之前 vs 切之後」的差。具體拆解：

- 把單一葉節點的 loss 用二階泰勒展開（→ 附錄 A.3 的二階形式 + Ch 6 完整推導），再對「葉節點輸出 $\gamma$」找最佳值，得到的「loss 減少量」剛好是 $\frac{1}{2} \cdot \frac{G^2}{H + \lambda}$
- 切之前：整個節點是單一葉子，loss 減少量 $= \frac{1}{2} \cdot \frac{G^2}{H + \lambda}$
- 切之後：變成左葉 + 右葉兩個獨立葉子，各自最佳化，total loss 減少量 $= \frac{1}{2}\bigl[\frac{G_L^2}{H_L + \lambda} + \frac{G_R^2}{H_R + \lambda}\bigr]$
- Gain $=$（切之後）$-$（切之前）= 切這一刀**多賺到**多少 loss 改善

所以 $G^2/(H+\lambda)$ 這個分式並不是憑空出現 ── 它就是「葉節點用 Newton step 找最佳輸出時」loss 能減多少。完整推導留到 Ch 6.4。

**挑切點的流程**：

1. 對每個特徵 $j$、每個候選閾值 $t$，計算對應的 Gain
2. 挑出 Gain 最大的 $(j, t)$ 作為這個節點的分裂規則
3. 如果最大 Gain $\le 0$（或低於某個門檻 $\epsilon$），就不切了，這個節點變葉節點

**Gain 結構的兩個重要性質**：

- **左右分得越乾淨，Gain 越大**：「乾淨」的精確定義是 $G_L$ 和 $G_R$ **異號且絕對值都大**（左邊樣本 $g_i$ 多為正、右邊多為負，或反之）。為什麼異號才算乾淨？因為 $G = G_L + G_R$ 是切前固定值，由代數恆等式 $G_L^2 + G_R^2 = G^2 - 2 G_L G_R$ 可知：$G_L G_R$ 越負（異號絕對值越大），$G_L^2 + G_R^2$ 越大、Gain 越大；若切完左右同號，$G_L G_R > 0$、$G_L^2 + G_R^2 < G^2$，Gain 反而可能變負（不值得切）。樹自然會挑「能把 $g_i$ 正負分開的特徵 + 閾值」。
- **小群體被 $\lambda$ 限縮**：分母 $H + \lambda$。
  - **$H$ 為什麼會小**：兩個原因。(a) 葉節點樣本數本來就少（$H$ 是 $h_i$ 加總，項目少自然小）；(b) 葉內樣本的 $p_i$ 多接近 $0$ 或 $1$（$h_i = p_i(1-p_i)$ 在邊界處趨於 $0$ ── 例如 $p = 0.99$ 時 $h \approx 0.01$，模型對該樣本「已經很確信」，沒什麼好學的，$h$ 給的資訊量小）
  - **為什麼要 $\lambda$ 拉住**：$H$ 趨近 $0$ 時 $G^2/H$ 趨近無窮，會把資訊量極少的小群體當成「Gain 巨大」誤判為值得切，等於放大訓練集的小樣本噪音。$\lambda > 0$ 把分母「從 $0$ 拉開」，限制這些「資訊不足葉子」對 Gain 的影響 ── 等價於說「不信任資訊量不夠的葉子」
  - **$\lambda$ 的來源與量綱**：$\lambda$ 是 **L2 正則化參數**（懲罰葉節點輸出 $\gamma$ 過大），Ch 6 推 Newton step 時會自然引入。它跟 $H$ 同量綱（$H$ 是 $h_i = p(1-p)$ 的加總，每筆 $h_i \in (0, 0.25]$）── 實務上常見 $\lambda \in [0, 10]$。具體調整屬於超參數調校：太小不夠正則化、太大會壓制有意義的切點

### 5.4 範例：單一切點的 Gain 計算

假設 4 筆樣本（2 正 2 負）落在某節點，當前所有人 $p_i = 0.3$。先算梯度：

| ID | $y$ | $g = p - y$ | $h = p(1-p)$ |
|---|---|---|---|
| A | $1$ | $-0.7$ | $0.21$ |
| B | $1$ | $-0.7$ | $0.21$ |
| C | $0$ | $+0.3$ | $0.21$ |
| D | $0$ | $+0.3$ | $0.21$ |

節點層級統計：$G = -1.4 + 0.6 = -0.8$、$H = 4 \times 0.21 = 0.84$。

假設考慮「特徵 $x_1$ 是否 $\ge t$」這個切點，剛好把 {A, B} 分到左、{C, D} 分到右：

- 左葉：$G_L = -1.4$、$H_L = 0.42$
- 右葉：$G_R = +0.6$、$H_R = 0.42$
- 守恆 $G_L + G_R = G$、$H_L + H_R = H$ 都成立 ✓

代入 Gain 公式（取 $\lambda = 0$）：

$$\text{Gain} = \dfrac{1}{2}\left[\dfrac{(-1.4)^2}{0.42} + \dfrac{0.6^2}{0.42} - \dfrac{(-0.8)^2}{0.84}\right] = \dfrac{1}{2}[4.667 + 0.857 - 0.762] \approx 2.381$$

拆解這個 $2.381$：「切之前」單一節點 loss 改善 $= \tfrac{1}{2}(0.762) = 0.381$；「切之後」左右兩葉各自最佳化的總 loss 改善 $= \tfrac{1}{2}(4.667 + 0.857) = 2.762$。Gain $\approx 2.381$ 就是「切這刀**多賺到**的 loss 改善」。

注意 $G_L = -1.4$、$G_R = +0.6$ 是異號 ── 5.3 講過的「左右分得越乾淨、Gain 越大」就是指這種情況：切完左邊全是「想往正類推」（$g < 0$）的樣本、右邊全是「想往負類推」（$g > 0$）的樣本，分得越乾淨平方和越大。

## Ch 6. 葉節點輸出：每片葉子要輸出多少

Ch 5 把樹的結構（哪裡切、切幾刀）決定好，每片葉子 $R_j$ 都對應一群樣本。現在問題剩下：**每片葉子應該輸出什麼值 $\gamma_j$？**

### 6.1 直覺先嘗試：能不能直接平均 $r$？

樹的訓練目標是 $r_i = y_i - p_i$（Ch 4 推得）。最直觀的想法：葉節點輸出 = 葉內目標值的平均：

$$\gamma_j \stackrel{?}{=} \frac{1}{|R_j|} \sum_{i \in R_j} r_i$$

這個答案**只在 squared error loss 下對**，logistic loss 下不對。為什麼？

對 squared error loss：要最小化 $\sum_{i \in R_j}(r_i - \gamma)^2$。對 $\gamma$ 微分為 $0$ 解出 $\gamma^* = \frac{1}{|R_j|}\sum r_i$（葉內 $r$ 平均）── 漂亮的閉式解。標準迴歸樹的演算法就是這樣訓練出來的。

但 GBDT 二元分類用的是 **logistic loss**。要最小化的是「葉內樣本加上 $\gamma$ 後的 logistic loss 總和」：

$$\text{Loss}_j(\gamma) = \sum_{i \in R_j} L\bigl(y_i,\, F_{m-1}(x_i) + \gamma\bigr)$$

對 $\gamma$ 微分（套 Ch 4.2 的單筆結果）：

$$\frac{d \text{Loss}_j}{d\gamma} = \sum_{i \in R_j} \bigl[\sigma\bigl(F_{m-1}(x_i) + \gamma\bigr) - y_i\bigr]$$

令這個為 $0$ 解 $\gamma$。問題：方程式裡每筆樣本的 $\sigma(F_{m-1}(x_i) + \gamma)$ 都是 $\gamma$ 的非線性函數，不同樣本還有不同的偏移 $F_{m-1}(x_i)$ ── 這個方程式**沒有閉式解**。

所以「平均 $r$」這個解法行不通。替代方案：**用二階泰勒展開把 loss 近似成 $\gamma$ 的二次式**，二次式有閉式最小值。這就是 Newton step。

### 6.2 葉節點 loss 的二階泰勒展開

要找「使葉節點 loss 最小的 $\gamma$」，但 logistic loss 對 $\gamma$ 非線性、沒有閉式解（6.1 已經看到）。策略：用二階泰勒近似（→ 附錄 A.3）把它換成 $\gamma$ 的二次式 ── 二次式有閉式最佳解。

**準備：$g_i$ 與 $h_i$ 的具體形式**

二階泰勒會用到兩個導數。對每筆樣本 $i$，這兩個導數都在「當前點」$F = F_{m-1}(x_i)$ 求值（同一個 $F_{m-1}$ ── Ch 5 決定樹結構時用過、現在 Ch 6 決定葉節點輸出又用一次；整個第 $m$ 輪共用）：

**一階導數 $g_i$**（Ch 4.2 已推）：

$$g_i = \dfrac{\partial L}{\partial F}\bigg|_{F = F_{m-1}(x_i)} = p_i - y_i, \qquad \text{其中 } p_i = \sigma\bigl(F_{m-1}(x_i)\bigr)$$

**二階導數 $h_i$**：對 logistic loss $L(y, F) = \log(1 + e^F) - yF$ 再微分一次。Ch 4.2 已知 $\partial L / \partial F = \sigma(F) - y$。對 $F$ 再微分一次（$y$ 不依賴 $F$，微分掉變 $0$）：

$$\frac{\partial^2 L}{\partial F^2} = \frac{d}{dF}\bigl[\sigma(F) - y\bigr] = \frac{d\sigma(F)}{dF} = \sigma(F)\bigl(1 - \sigma(F)\bigr)$$

最後一步用附錄 A.2 推過的 sigmoid 微分。在當前點求值：

$$h_i = \sigma\bigl(F_{m-1}(x_i)\bigr) \cdot \bigl[1 - \sigma(F_{m-1}(x_i))\bigr] = p_i (1 - p_i)$$

這就是 Ch 5 採信過的 $h_i$ 公式來源。注意 $h_i > 0$ 恆成立（因為 $p_i \in (0, 1)$ 使 $p_i$ 和 $1-p_i$ 都嚴格正），這點 6.3 求最小值時會用到。

**葉內二階泰勒展開**

有了 $g_i, h_i$ 的具體值，對每筆樣本 $i \in R_j$，把 $L(y_i, F + \gamma)$ 在 $F = F_{m-1}(x_i)$ 處展開（保留到二階；$\gamma^3$ 以上丟掉，因為 $\nu \in [0.05, 0.3]$ 的 learning rate 把每棵樹的修正幅度壓小，$\gamma$ 通常落在二階近似的有效範圍內）：

$$L\bigl(y_i,\, F_{m-1}(x_i) + \gamma\bigr) \;\approx\; L\bigl(y_i, F_{m-1}(x_i)\bigr) + g_i \gamma + \tfrac{1}{2} h_i \gamma^2$$

（這裡的 $\gamma$ 對應附錄 A.3 中的「偏移量 $\delta$」── 都是「從當前 $F$ 加上去的修正量」，只是符號不同。）

**葉內加總**：對 $R_j$ 內所有樣本加總二階近似：

$$\text{Loss}_j(\gamma) \;\approx\; \underbrace{\sum_{i \in R_j} L\bigl(y_i, F_{m-1}(x_i)\bigr)}_{\text{常數 } C_j,\;\text{跟 } \gamma \text{ 無關}} + \left(\sum_{i \in R_j} g_i\right) \gamma + \frac{1}{2}\left(\sum_{i \in R_j} h_i\right) \gamma^2$$

引入葉節點統計量（Ch 5 用過的記號）：

$$G_j = \sum_{i \in R_j} g_i, \qquad H_j = \sum_{i \in R_j} h_i$$

整體 loss 變成 $\gamma$ 的**二次式**：

$$\text{Loss}_j(\gamma) \approx C_j + G_j \gamma + \tfrac{1}{2} H_j \gamma^2$$

### 6.3 求解 $\gamma$：Newton step

二次式的最小值用一階條件（→ 附錄 A.1）找。先確認凸性：二次項係數是 $\tfrac{1}{2} H_j$，而 $H_j = \sum_{i \in R_j} h_i > 0$（6.2 推過每筆 $h_i > 0$，加總仍嚴格正），所以 $\text{Loss}_j(\gamma)$ 對 $\gamma$ 是凸二次式 ── 一階條件給出全局最小值，不會誤抓到極大值。

對 $\gamma$ 微分：

$$\frac{d \text{Loss}_j}{d \gamma} = G_j + H_j \gamma$$

（$C_j$ 是常數微分掉變 $0$；$\tfrac{1}{2} H_j \gamma^2$ 微分得 $H_j \gamma$。）

令導數 $= 0$：

$$G_j + H_j \gamma = 0$$

解出：

$$\boxed{\;\gamma_j^* = -\dfrac{G_j}{H_j}\;}$$

這就是 **Newton step** 在葉節點上的應用 ── 對應附錄 A.3 的 $\delta^* = -B/A$（這裡 $A = H_j$、$B = G_j$）。

### 6.4 加 L2 正則化 + 回填 Ch 5 的 Gain 公式

**為什麼需要 L2**：上式 $\gamma_j^* = -G_j / H_j$ 在 $H_j$ 很小時會爆衝（Ch 5 討論過 ── 葉節點樣本少、或 $p$ 在邊界使 $h = p(1-p) \to 0$）。爆衝代表「對資訊量極少的葉節點，我們給了一個極大的修正」，會把訓練資料噪音放大。

實務上 XGBoost / LightGBM 在「葉節點 loss」加上 L2 正則化項，懲罰 $\gamma$ 過大。具體形式選 $\tfrac{1}{2}\lambda\gamma^2$（而非 $\lambda\gamma^2$）有兩個原因：(a) 跟原本的二次項 $\tfrac{1}{2}H_j\gamma^2$ 形式一致，合併後二次項係數變成 $\tfrac{1}{2}(H_j + \lambda)$，整體公式漂亮；(b) 對 $\gamma$ 微分後 $\tfrac{1}{2}$ 跟 $\gamma^2$ 的 $2$ 相消，得到 $\lambda \gamma$，跟一階導數的單位一致。

$$\text{Loss}_j(\gamma) + \tfrac{1}{2} \lambda \gamma^2 \;\approx\; C_j + G_j \gamma + \tfrac{1}{2}(H_j + \lambda) \gamma^2$$

$\lambda$ 跟 $H_j$ 同量綱（$h_i = p(1-p) \in (0, 0.25]$，所以 $H_j$ 約是「葉內樣本數 $\times 0.25$ 以下」的數量級；$\lambda$ 一般設成跟單筆 $h$ 或數筆 $h$ 同等大小，例如 $\lambda \in [0, 10]$，具體靠超參數調校）。

對 $\gamma$ 求最佳值，同樣套一階條件：

$$\frac{d}{d\gamma}\bigl[C_j + G_j \gamma + \tfrac{1}{2}(H_j + \lambda)\gamma^2\bigr] = G_j + (H_j + \lambda)\gamma = 0$$

解出（這是 GBDT 葉節點輸出的**最終公式**）：

$$\boxed{\;\gamma_j^* = -\dfrac{G_j}{H_j + \lambda}\;}$$

$\lambda$ 在分母累加 ── $H_j$ 小（資訊量少）時 $\lambda$ 主導，把 $\gamma$ 拉小、避免爆衝；$H_j$ 大（資訊量充足）時 $\lambda$ 影響小，$\gamma$ 趨近 $-G_j/H_j$。

**回填 Ch 5 的 Gain 公式**：把 $\gamma_j^* = -G_j/(H_j + \lambda)$ 代回「原 loss + L2 正則化項」這個整體目標（注意：$\text{Loss}_j(\gamma)$ 始終指原始 logistic loss、不含 L2；正則項 $\tfrac{1}{2}\lambda\gamma^2$ 是另外加上去的）：

$$\text{Loss}_j(\gamma_j^*) + \tfrac{1}{2}\lambda (\gamma_j^*)^2 = C_j + G_j \cdot \left(-\dfrac{G_j}{H_j+\lambda}\right) + \tfrac{1}{2}(H_j + \lambda) \cdot \left(-\dfrac{G_j}{H_j+\lambda}\right)^2$$

逐項化簡：

- 第二項（一次項代入）：$G_j \cdot \bigl(-\frac{G_j}{H_j+\lambda}\bigr) = -\dfrac{G_j^2}{H_j+\lambda}$
- 第三項（二次項代入 + 平方）：$\tfrac{1}{2}(H_j+\lambda) \cdot \dfrac{G_j^2}{(H_j+\lambda)^2}$。分母 $(H_j+\lambda)^2$ 跟分子前的因子 $(H_j+\lambda)$ 約掉一個，剩 $\tfrac{1}{2} \cdot \dfrac{G_j^2}{H_j+\lambda}$

合起來：

$$\text{Loss}_j(\gamma_j^*) + \tfrac{1}{2}\lambda (\gamma_j^*)^2 = C_j - \frac{G_j^2}{H_j+\lambda} + \frac{1}{2} \cdot \frac{G_j^2}{H_j+\lambda} = C_j - \frac{1}{2} \cdot \frac{G_j^2}{H_j+\lambda}$$

跟「不做任何修正」的 baseline $C_j$ 相比（$\gamma = 0$ 時，原 loss 就是 $C_j$、L2 項是 $0$），**葉節點用 Newton step 最佳化後 loss 減少了**：

$$\Delta_j = C_j - \bigl[\text{Loss}_j(\gamma_j^*) + \tfrac{1}{2}\lambda (\gamma_j^*)^2\bigr] = \frac{1}{2} \cdot \frac{G_j^2}{H_j+\lambda}$$

這就是 Ch 5 直覺段落講的「該葉子用 Newton step 找到最佳輸出時，能讓 loss 減少的量」。

**Gain 公式組成**：套到「切之前 vs 切之後」：

- **切之前**：把這個節點當成單一葉子（不再切），它的最佳輸出讓 loss 減少 $\Delta_{\text{node}} = \tfrac{1}{2} \cdot \tfrac{G^2}{H+\lambda}$（用上式，把 $G_j, H_j$ 換成節點整體的 $G, H$）
- **切之後**：變成左葉 + 右葉兩個獨立葉子，各自最佳化，loss 減少量總和 $= \Delta_L + \Delta_R = \tfrac{1}{2}\bigl[\tfrac{G_L^2}{H_L+\lambda} + \tfrac{G_R^2}{H_R+\lambda}\bigr]$
- **Gain = 切之後 − 切之前**（也就是「多賺到的 loss 改善」）：

$$\text{Gain} = \dfrac{1}{2}\left[\dfrac{G_L^2}{H_L + \lambda} + \dfrac{G_R^2}{H_R + \lambda} - \dfrac{G^2}{H + \lambda}\right]$$

跟 Ch 5 採信的 Gain 公式完全一致 ── 推導結束。樹結構與葉節點輸出兩個剩餘問題都解完。

### 6.5 範例：Newton step vs 直觀平均 $r$

6.1 講過「直接平均 $r$ 在 logistic loss 下不對」。這個範例把差距具體化。

假設某葉節點有 3 個樣本，剛好處於模型「不同確信程度」的點上：

| ID | $y$ | $F_{m-1}$ | $p = \sigma(F)$ | $r = y - p$ | $h = p(1-p)$ |
|---|---|---|---|---|---|
| A | $1$ | $-2$ | $0.119$ | $+0.881$ | $0.105$ |
| B | $1$ | $\phantom{-}0$ | $0.500$ | $+0.500$ | $0.250$ |
| C | $0$ | $+2$ | $0.881$ | $-0.881$ | $0.105$ |

葉內統計：$\sum r = +0.5$、$\sum h = 0.460$。三種 $\gamma$ 計算對比：

| 方法 | 公式 | $\gamma$ |
|---|---|---|
| 直觀平均 $r$（6.1 的 strawman） | $\frac{0.5}{3}$ | $\approx 0.167$ |
| **Newton step**（$\lambda = 0$） | $-G/H = \frac{0.5}{0.46}$ | $\approx 1.087$ |
| Newton + $\lambda = 1$ | $\frac{0.5}{1.46}$ | $\approx 0.342$ |

Newton step 是直觀平均的 **約 6.5 倍**。原因：A 跟 C 的 $p$ 在邊界（$\sigma$ 接近飽和），$h = 0.105$ 很小 ── 模型對它們「已經很確信」，logistic loss 在這兩點對 $\gamma$ 的曲率低，影響有限。$|r|$ 大不代表 logistic loss 下「值得修正得多」，要看 $h$ 加權後的結果。直觀平均把 A、B、C 等權對待，等於忽略 $h$ 的差異，所以偏離最佳解。

再看 $\lambda$ 的效果：加 $\lambda = 1$ 後 $\gamma$ 從 $1.087$ 拉回 $0.342$。這個葉子只有 3 個樣本、$H = 0.46$ 不算大，$\lambda$ 把它拉回更保守的修正幅度，避免「3 個樣本的訊號被放大成 $\gamma > 1$ 的爆衝」。

## Ch 7. 預測流程：把 1–6 串起來

訓練完成後，模型由「$F_0$ 一個數字 + $M$ 棵訓練好的樹」組成。每棵樹的內部結構（切分規則 + 葉節點輸出 $\gamma_j^*$）都已確定。對新樣本 $x$，要算出 $p(x)$ 的步驟如下。

### 7.1 完整步驟

**Step 0：拿到輸入** $x \in \mathbb{R}^d$。

**Step 1：算 raw score $F(x)$**

從 $F_0$ 起算（Ch 3 算好的常數）：

$$F(x) = F_0 + \nu \cdot \sum_{m=1}^{M} f_m(x)$$

每一項 $f_m(x)$ 怎麼得？對第 $m$ 棵樹，從根節點走起：

1. 在每個內部節點，根據 $x$ 的特徵值對照分裂規則（Ch 5 訓練時決定的「特徵 $j$ 是否 $\ge$ 閾值 $t$」），決定往左子節點或右子節點走
2. 一路走到某個葉節點 $R_{j(m, x)}$（這個 $j(m, x)$ 表示「$x$ 在第 $m$ 棵樹落到哪片葉子」）
3. 該葉節點存的常數就是 $f_m(x) = \gamma_{j(m, x)}^*$（Ch 6 訓練時用 Newton step 算好的）

把每棵樹的輸出 $f_m(x)$ 全部加總、乘上 learning rate $\nu$、加上 $F_0$，就是 $F(x)$。

**Step 2：sigmoid 變機率**

$$p(x) = \sigma\bigl(F(x)\bigr) = \dfrac{1}{1 + e^{-F(x)}}$$

得到的 $p(x) \in [0, 1]$ 就是「$x$ 屬於正類」的預測機率。

**Step 3（可選）：硬決策**

如果應用需要二元類別輸出：

$$\hat y = \mathbb{1}\bigl[p(x) \ge 0.5\bigr] = \mathbb{1}\bigl[F(x) \ge 0\bigr]$$

兩個條件等價（Ch 1.4 已說過：sigmoid 在 $F = 0$ 處剛好 $= 0.5$）。

### 7.2 全文邏輯鏈一覽

整份手冊推導出的所有元件：

| 元件 | 結果 | 從哪來 |
|---|---|---|
| 機率怎麼來 | $p = \sigma(F)$ | Ch 1（GBDT 兩階段設計，$F$ 是 log-odds） |
| $F$ 怎麼組成 | $F = F_0 + \nu \sum_{m=1}^M f_m(x)$ | Ch 2（加法模型 + log-odds 加法 = odds 乘法） |
| $F_0$ 怎麼來 | $F_0 = \log\frac{\bar y}{1-\bar y}$ | Ch 3（最小化整體 loss 解出來） |
| 樹要學什麼 | $r = y - p$（pseudo-residual = 負梯度） | Ch 4（一階泰勒近似 + functional gradient descent） |
| 樹結構怎麼決定 | greedy 生長 + Gain 挑切點 | Ch 5（每個內部節點挑 Gain 最大的切點） |
| 葉節點輸出 | $\gamma_j^* = -\frac{G_j}{H_j + \lambda}$ | Ch 6（二階泰勒 + Newton step + L2 正則化） |
| 預測流程 | 上面三步驟 | 本章 |

### 7.3 一句話總結

> **GBDT 二元分類的 raw output $F(x)$ 是 log-odds；sigmoid 後才是機率 $p(x)$。$F$ 由 $F_0$（資料的 prior log-odds）加上一連串樹的輸出累積而成；每棵樹學的是「當前模型對訓練樣本的 pseudo-residual $r = y - p$」，樹的結構靠 Gain 量化切的好壞、葉節點輸出靠 Newton step 找最佳 $\gamma$。**

注意「樹學 $r$」這個說法是指「$r$ 是樹的訓練目標」── 不代表「樹的輸出 $= r$」。樹的實際輸出是把樣本按特徵分群、每群吐一個用 Newton step 算出的常數 $\gamma_j^* = -G_j / (H_j + \lambda)$；這些常數共同對訓練資料的 $\{r_i\}$ 做最佳逼近。

具體走查見 Ch 8 ── 用 6 個客戶的 toy 資料跑完 $F_0$ → 3 棵樹，每一步都標清楚對應哪個公式。

## Ch 8. 完整走查：6 客戶從 prior 到 $p(x)$

用一份小資料把 Ch 1–6 的元件全部跑一遍。**每一步都註明對應哪一章的公式**，把抽象推導變回具體計算。為了讓樹結構手算可行，刻意把資料縮到 6 筆、2 特徵；$\lambda = 0$（無正則化）、$\nu = 0.1$（learning rate）。

### 8.1 資料與初始狀態

| ID | tenure（月） | age（歲） | $y$ |
|---|---|---|---|
| 1 | $\phantom{0}2$ | $25$ | $0$ |
| 2 | $24$ | $40$ | $1$ |
| 3 | $\phantom{0}6$ | $31$ | $0$ |
| 4 | $18$ | $37$ | $1$ |
| 5 | $\phantom{0}4$ | $28$ | $0$ |
| 6 | $12$ | $42$ | $0$ |

$\bar y = 2/6 \approx 0.333$。

**算 $F_0$**（Ch 3.4）：

$$F_0 = \log \dfrac{0.333}{0.667} = \log(0.5) \approx -0.693$$

**初始預測機率**（Ch 1.1）：所有樣本 $p_{i,0} = \sigma(-0.693) \approx 0.333$。模型還沒看任何特徵，每個樣本都猜 33%。

### 8.2 第 1 棵樹

**算 $g_i, h_i$**（Ch 4.2、Ch 6.2）。因為所有 $p_{i,0} = 0.333$，分兩類：

| 類別 | $y$ | $g = p - y$ | $h = p(1-p)$ |
|---|---|---|---|
| 正樣本（#2, #4） | $1$ | $-0.667$ | $0.222$ |
| 負樣本（#1, #3, #5, #6） | $0$ | $\phantom{-}0.333$ | $0.222$ |

**根節點統計**：

$$G = 2(-0.667) + 4(0.333) = 0, \qquad H = 6 \times 0.222 = 1.333$$

（$G = 0$ 不是巧合 ── 在 $\bar p = \bar y$ 的狀態下 $\sum g = n\bar p - n\bar y = 0$。下一棵樹之後 $\bar p$ 跟 $\bar y$ 不再相等，$G$ 就會偏離 $0$。）

**掃過候選切點算 Gain**（Ch 5.3 公式，$\lambda = 0$）：

| 切點 | 左葉 | 右葉 | $G_L$ | $H_L$ | $G_R$ | $H_R$ | Gain |
|---|---|---|---|---|---|---|---|
| tenure ≥ 12 | {1,5,3} 全負 | {6,4,2} 1負2正 | $\phantom{-}1.000$ | $0.667$ | $-1.000$ | $0.667$ | $1.500$ |
| **tenure ≥ 18** | **{1,5,3,6} 全負** | **{4,2} 全正** | $\phantom{-}\mathbf{1.333}$ | $\mathbf{0.889}$ | $\mathbf{-1.333}$ | $\mathbf{0.444}$ | $\mathbf{3.000}$ ★ |
| tenure ≥ 24 | {1,5,3,6,4} 4負1正 | {2} 1正 | $\phantom{-}0.667$ | $1.111$ | $-0.667$ | $0.222$ | $1.200$ |
| age ≥ 37 | {1,5,3} 全負 | {4,2,6} 1負2正 | $\phantom{-}1.000$ | $0.667$ | $-1.000$ | $0.667$ | $1.500$ |
| age ≥ 42 | {1,5,3,4,2} 3負2正 | {6} 1負 | $-0.333$ | $1.111$ | $\phantom{-}0.333$ | $0.222$ | $0.300$ |

**$G_L, H_L, G_R, H_R$ 怎麼填？以 ★（tenure ≥ 18）為例**：這四欄不是新東西，就是把上面 $g_i, h_i$ 那張表按「左葉 / 右葉」分組後**各自加總**（Ch 5.3 定義 $G_L = \sum_{i \in \text{left}} g_i$、$H_L = \sum_{i \in \text{left}} h_i$，右葉同理）。tenure ≥ 18 這刀把 {1,5,3,6} 分到左、{4,2} 分到右：

- **左葉 {1,5,3,6}** 全是負樣本，每筆 $g = +0.333$、$h = 0.222$（4 筆相同）：

$$G_L = 0.333 \times 4 = 1.333, \qquad H_L = 0.222 \times 4 = 0.889$$

- **右葉 {4,2}** 全是正樣本，每筆 $g = -0.667$、$h = 0.222$（2 筆相同）：

$$G_R = (-0.667) \times 2 = -1.333, \qquad H_R = 0.222 \times 2 = 0.444$$

驗守恆關係（Ch 5.3）：$G_L + G_R = 1.333 + (-1.333) = 0 = G$、$H_L + H_R = 0.889 + 0.444 = 1.333 = H$ ✓ ── 樣本只是被分組、沒有消失。表中其他切點的 $G_L, H_L, G_R, H_R$ 算法完全一樣，差別只在哪些樣本被分到左、哪些到右。

**tenure ≥ 18** 唯一最大，取為第 1 刀。注意：age ≥ 42 雖然「孤立 #6」（#6 是負樣本但 age 比所有正樣本還大，看似「異常」），但這刀 Gain 只有 $0.3$ ── 因為孤立 #6 後左葉變成 {1, 5, 3, 4, 2}（3 負 + 2 正混雜），$G_L$ 接近 $0$、$G_L^2$ 很小，沒幫忙分正負類。Gain 公式自動避開這個「直覺上有道理但對 loss 沒幫助」的切點。

**驗算 Gain $= 3.0$**：

$$\text{Gain} = \dfrac{1}{2}\left[\dfrac{1.333^2}{0.889} + \dfrac{(-1.333)^2}{0.444} - \dfrac{0^2}{1.333}\right] = \dfrac{1}{2}[2.0 + 4.0 - 0] = 3.0 \;\checkmark$$

**繼續切？** 左葉 4 個樣本 $y$ 全為 $0$、右葉 2 個樣本 $y$ 全為 $1$ ── 兩葉內樣本標籤已單一。同葉內 $g_i$ 都同值（左葉全 $+0.333$、右葉全 $-0.667$），再切下去 $G_L'^2 + G_R'^2$ 不會大於現在的 $G_L^2 + G_R^2$，Gain $= 0$（甚至負）。樹 1 蓋完，2 葉。

**葉節點輸出**（Ch 6.4，$\lambda = 0$）：

$$\gamma_L^* = -\dfrac{G_L}{H_L} = -\dfrac{1.333}{0.889} = -1.500$$

$$\gamma_R^* = -\dfrac{G_R}{H_R} = -\dfrac{-1.333}{0.444} = +3.000$$

**更新 $F$**（Ch 2.1，$\nu = 0.1$）：

- **L 葉（4 個負樣本：#1, #3, #5, #6）**：$F_1 = -0.693 + 0.1 \times (-1.500) = -0.843$，$p_1 = \sigma(-0.843) \approx 0.301$
- **R 葉（2 個正樣本：#2, #4）**：$F_1 = -0.693 + 0.1 \times 3.000 = -0.393$，$p_1 = \sigma(-0.393) \approx 0.403$

樹 1 蓋完，兩群分開了 ── L 群從 $0.333$ 微降到 $0.301$，R 群從 $0.333$ 拉高到 $0.403$。

### 8.3 第 2 棵樹

現在 $p_i$ 不再一致，要按組重算 $g_i, h_i$：

| 組 | 樣本 | $p$ | $g = p - y$ | $h = p(1-p)$ |
|---|---|---|---|---|
| L 群（4 個負樣本） | #1, #3, #5, #6 | $0.301$ | $\phantom{-}0.301$ | $0.210$ |
| R 群（2 個正樣本） | #2, #4 | $0.403$ | $-0.597$ | $0.241$ |

**根節點統計**：

$$G = 4(0.301) + 2(-0.597) = 1.204 - 1.194 = 0.010$$

$$H = 4(0.210) + 2(0.241) = 0.840 + 0.482 = 1.322$$

（$G$ 不再剛好 $= 0$，因為 $\bar p$ 已偏離 $\bar y$。）

**最佳切點仍是 tenure ≥ 18**（資料未變、最強的特徵還是 tenure；其他候選 Gain 都更小，略）：

- L 葉：$G_L = 4(0.301) = 1.204$，$H_L = 4(0.210) = 0.840$
- R 葉：$G_R = 2(-0.597) = -1.194$，$H_R = 2(0.241) = 0.482$

$$\text{Gain} = \dfrac{1}{2}\left[\dfrac{1.204^2}{0.840} + \dfrac{(-1.194)^2}{0.482} - \dfrac{0.010^2}{1.322}\right] \approx \dfrac{1}{2}[1.726 + 2.957 - 0] \approx 2.343$$

Gain 比樹 1 的 $3.0$ 小（$2.343$）── 因為樹 1 已經把這條訊息榨了一部分。一個量化跡象：根節點 $G$ 從樹 1 前的 $0$ 變成 $0.010$（接近但不等於 $0$，因為兩群 $p$ 已偏離 $\bar y$），整體的「正類傾向梯度」被樹 1 的兩葉吸收了一部分，殘留信號自然較弱。

**葉節點輸出**：

$$\gamma_L^* = -\dfrac{1.204}{0.840} \approx -1.433, \qquad \gamma_R^* = -\dfrac{-1.194}{0.482} \approx +2.479$$

**更新 $F$**：

- L 群：$F_2 = -0.843 + 0.1 \times (-1.433) = -0.986$，$p_2 = \sigma(-0.986) \approx 0.272$
- R 群：$F_2 = -0.393 + 0.1 \times 2.479 = -0.145$，$p_2 = \sigma(-0.145) \approx 0.464$

R 群已經逼近 0.5 門檻。

### 8.4 第 3 棵樹與跨門檻

重算 $g, h$：

| 組 | $p$ | $g$ | $h$ |
|---|---|---|---|
| L 群（4 個負） | $0.272$ | $\phantom{-}0.272$ | $0.198$ |
| R 群（2 個正） | $0.464$ | $-0.536$ | $0.249$ |

$$\gamma_L^* = -\dfrac{1.088}{0.792} \approx -1.374, \qquad \gamma_R^* = -\dfrac{-1.072}{0.498} \approx +2.153$$

更新 $F$：

- L 群：$F_3 = -0.986 + 0.1 \times (-1.374) = -1.123$，$p_3 \approx 0.245$
- **R 群：$F_3 = -0.145 + 0.1 \times 2.153 = +0.070$，$p_3 \approx 0.518$ ← 跨過 $0.5$ 門檻**

R 群的正樣本（#2, #4）終於在第 3 棵樹後 $p > 0.5$，模型開始把它們判為正類。

**為什麼非要 3 棵樹才能跨門檻？** R 群的 $F$ 從 $-0.693$ 推到 $> 0$ 需要累積修正 $> 0.693$。每棵樹貢獻 $\nu \cdot \gamma_R \approx 0.1 \times \{3.0, 2.5, 2.2\} = 0.30, 0.25, 0.22$（樹愈後面 $\gamma$ 愈小，因為前一棵已經把訊號榨走一部分）。3 棵累計 $0.77 > 0.693$，剛好跨過。這體現了 learning rate 的「shrinkage」效應（Ch 2.2）── 小步多走，不一棵樹爆衝。

**完整進程摘要**：

| 階段 | L 群 $F$ | L 群 $p$ | R 群 $F$ | R 群 $p$ | R 群 $\hat y$ |
|---|---|---|---|---|---|
| $F_0$（無樹） | $-0.693$ | $0.333$ | $-0.693$ | $0.333$ | $0$ |
| 樹 $1$ | $-0.843$ | $0.301$ | $-0.393$ | $0.403$ | $0$ |
| 樹 $2$ | $-0.986$ | $0.272$ | $-0.145$ | $0.464$ | $0$ |
| **樹 $3$** | $-1.123$ | $0.245$ | $\phantom{-}0.070$ | $\mathbf{0.518}$ | $\mathbf{1}$ ★ |

**實務上什麼時候停？**

- **樹數上限** $M$：超參數，比方 100、500 棵
- **early stopping**：用 validation set 監控；連續 $k$ 輪 metric 沒進步就停
- **Gain 門檻**：所有候選切點 Gain 都低於 $\epsilon$ 時停（本例已經出現「樹 3 後 Gain 持續下降」的趨勢，再多幾棵 Gain 就接近 0）

本例 toy 資料量太小、結構太單純，3 棵樹剛好夠跨門檻。真實資料通常需要 $10^2 \sim 10^3$ 棵。

## 附錄 A：微積分小複習

本文假設你學過微積分但不熟。下面 3 個工具會在主章節用到，每次需要時主章節直接寫「→ 見附錄 A.N」。

### A.1 一階條件：可微凸函數的最小值在「導數 = 0」處

**陳述**：若 $f(\gamma)$ 是可微的凸函數，則它的最小值點 $\gamma^*$ 滿足 $f'(\gamma^*) = 0$。

**直覺**：
- 在 $\gamma^*$ 左邊（往負方向走），$f$ 還在下降，所以 $f' < 0$
- 在 $\gamma^*$ 右邊（往正方向走），$f$ 已經上升，所以 $f' > 0$
- 剛好在 $\gamma^*$：分界點，$f' = 0$

**為什麼「凸性」是關鍵**：對一般函數，「導數 = 0」的點可能是「局部最小」、「局部最大」、或「鞍點」三種狀況，無法區分。只有當函數是凸的，「導數 = 0」才能保證該點是「全局最小」。logistic loss 對 $\gamma$ 是凸函數（後面 Ch 6 推導時會看到它的二階導數始終為正），所以我們直接拿一階條件當「最小值」的充要條件，不需要再驗證二階。

**怎麼用**：要找「使 $\sum L(\gamma)$ 最小的那個 $\gamma$」，就解方程式

$$\frac{d}{d\gamma}\sum_{i} L(y_i, \gamma) = 0$$

得到的解就是最佳 $\gamma$。

### A.2 chain rule 與 sigmoid 微分

**chain rule（合成函數微分）**：

$$\frac{d}{dF}\,g\bigl(h(F)\bigr) = g'\bigl(h(F)\bigr) \cdot h'(F)$$

直白翻譯：先對外層函數 $g$ 微分（把中間層的輸出 $h(F)$ 當成一個新變數 $u$，算 $g'(u)$ 再代回 $u = h(F)$），再乘以中間層 $h$ 對 $F$ 的微分。

**例子**：算 $\frac{d}{dF}\log(1 + e^F)$。

- 外層 $g(u) = \log u$，導數 $g'(u) = 1/u$
- 中間層 $h(F) = 1 + e^F$，導數 $h'(F) = e^F$
- 套 chain rule：

$$\frac{d}{dF}\log(1 + e^F) = \frac{1}{1 + e^F} \cdot e^F = \frac{e^F}{1 + e^F}$$

**sigmoid 函數的兩個等價形式**：

$$\sigma(F) = \frac{1}{1 + e^{-F}} = \frac{e^F}{1 + e^F}$$

兩種寫法數值相同。要從第一式得到第二式：分子分母同乘 $e^F$ ── 分子變 $e^F$；分母 $(1 + e^{-F}) \cdot e^F = e^F + 1$；合起來就是 $\dfrac{e^F}{1 + e^F}$。論文與教材兩種寫法都常見，看到不要打結。

**串起來的結論**：上面 $\log(1+e^F)$ 的微分結果剛好就是 sigmoid：

$$\boxed{\;\frac{d}{dF}\log(1 + e^F) = \sigma(F)\;}$$

為什麼這個事實重要：Ch 4 要對 logistic loss 微分，而 logistic loss 的單筆樣本形式就是 $L = \log(1 + e^F) - yF$ ── 上面這個結果是 Ch 4 推導第一步的關鍵。

**sigmoid 自己的微分**（Ch 6 Newton step 會用到）：

$$\frac{d\sigma(F)}{dF} = \sigma(F)\bigl(1 - \sigma(F)\bigr)$$

**怎麼驗**：把 $\sigma(F) = (1 + e^{-F})^{-1}$ 看成「外層 $u^{-1}$、中間層 $1 + e^{-F}$」的合成函數。

- 外層 $u^{-1}$ 的導數是 $-u^{-2}$
- 中間層 $1 + e^{-F}$ 的導數是 $-e^{-F}$
- 套 chain rule：

$$\frac{d\sigma}{dF} = -\frac{1}{(1+e^{-F})^2} \cdot (-e^{-F}) = \frac{e^{-F}}{(1+e^{-F})^2}$$

為了讓分子和分母有公因子可以約分，把分子 $e^{-F}$ 拆成 $(1 + e^{-F}) - 1$：

$$= \frac{(1+e^{-F}) - 1}{(1+e^{-F})^2} = \frac{1}{1+e^{-F}} - \frac{1}{(1+e^{-F})^2} = \sigma(F) - \sigma(F)^2 = \sigma(F)\bigl(1 - \sigma(F)\bigr)$$

### A.3 泰勒展開與 Newton step

**泰勒展開（保留到二階）**：把任何夠光滑的函數 $L$ 在某一點 $F_0$ 附近用多項式近似（「夠光滑」+「$\delta$ 夠小」是這個近似精確的隱含前提；GBDT 用 learning rate $\nu \in [0.05, 0.3]$ 把每棵樹的修正幅度壓小，使得 $\delta$ 通常落在這個近似的有效範圍內）：

$$L(F_0 + \delta) \;\approx\; L(F_0) + L'(F_0)\,\delta + \tfrac{1}{2}L''(F_0)\,\delta^2 + \cdots$$

其中：
- $\delta$ 是「從 $F_0$ 偏移的量」
- $L'(F_0)$、$L''(F_0)$ 是 $L$ 在 $F_0$ 的一階、二階導數
- $\cdots$ 是 $\delta^3$ 以上的高次項，當 $\delta$ 小時很小

**每一項的意義**：
- **零階項** $L(F_0)$：在起點 $F_0$ 的 loss 數值，跟 $\delta$ 無關
- **一階項** $L'(F_0)\,\delta$：告訴你「$\delta$ 往哪個正負方向走會讓 loss 變小」
- **二階項** $\tfrac{1}{2}L''(F_0)\,\delta^2$：曲率資訊。$\delta$ 拉太遠時這個正二次項變主導，避免「步長太大反而讓 loss 上升」

如果只用一階項：永遠覺得「往負梯度方向越遠越好」，沒終點。加入二階項：才有「步長多大才剛好」的答案。

**Newton step 的核心想法**：原始 loss $L$ 可能難以直接最小化（沒有封閉解）。Newton step 的策略是 ── 把 $L$ 暫時替換成它的「二階泰勒近似」這個 mini-loss：

$$\widetilde{L}(\delta) = L(F_0) + L'(F_0)\,\delta + \tfrac{1}{2}L''(F_0)\,\delta^2$$

然後對這個 mini-loss 求最佳 $\delta$。$\widetilde{L}$ 對 $\delta$ 是二次式 $\tfrac{A}{2}\delta^2 + B\,\delta + C$（其中 $A = L''(F_0)$、$B = L'(F_0)$、$C = L(F_0)$）。

二次式的最小值（套 A.1 的一階條件，對 $\delta$ 微分為 0）：

$$\frac{d\widetilde{L}}{d\delta} = A\,\delta + B = 0 \;\;\Longrightarrow\;\; \delta^* = -\frac{B}{A} = -\frac{L'(F_0)}{L''(F_0)}$$

這就是 **Newton step**：在 $F_0$ 處用二次式近似 loss，得到的下一步建議。

**為什麼是 $-B/A$ 不是 $-B/(2A)$**：很多人記得「$ax^2 + bx + c$ 的頂點在 $-b/(2a)$」，這裡看起來少了一個 2。原因：我們二次項的係數是 $A/2$ 不是 $A$，代進頂點公式 $-B/(2 \cdot \tfrac{A}{2}) = -B/A$，剛好把 2 抵消掉。Newton step 公式裡那個 $\tfrac{1}{2}$ 是泰勒展開原生帶的，不是額外加的。

（如果你沒記得頂點公式也沒關係：直接套 A.1 的一階條件對 $\widetilde{L}$ 微分為 0，就會得到 $A\delta + B = 0$，解出 $\delta^* = -B/A$，跟頂點公式殊途同歸。）
