│ 1. build_scoring_dataset(feature_table, parameters) -> pd.DataFrame          │
│                                                                              │
│ - 從 parameters["inference"]["snap_dates"] 篩選要評分的 snap_date            │
│ - 從 parameters["inference"]["products"] 取得 22 個產品列表                  │
│ - 取 feature_table 中目標 snap_date 的不重複 (snap_date, cust_id)            │
│ - Cross-join 產品列表，產生 (snap_date, cust_id, prod_name) 笛卡爾積         │
│ - Left-join 回 feature_table 附上特徵欄位                                    │
│ - 輸出包含 snap_date, cust_id, prod_name 及所有特徵欄位                      │
│                                                                              │
│ 2. apply_preprocessor(scoring_dataset, preprocessor) -> pd.DataFrame         │
│                                                                              │
│ - 使用 preprocessor["drop_columns"] 移除欄位（errors="ignore"，因 inference  │
│ 無 label）                                                                   │
│ - 用 preprocessor["category_mappings"] 對 categorical_columns 做編碼         │
│ - 確保輸出欄位順序與 preprocessor["feature_columns"] 完全一致                │
│ - 驗證：若有缺少的特徵欄位，拋出明確錯誤                                     │
│                                                                              │
│ 3. predict_scores(model, X_score, scoring_dataset) -> pd.DataFrame           │
│                                                                              │
│ - 呼叫 model.predict(X_score) 取得機率分數                                   │
│ - 回傳 DataFrame：[snap_date, cust_id, prod_code, score]                     │
│ - prod_name 直接作為 prod_code（Strategy 1）                                 │
│                                                                              │
│ 4. rank_predictions(score_table, parameters) -> pd.DataFrame                 │
│                                                                              │
│ - 按 (snap_date, cust_id) 分組                                               │
│ - 依 score 降序排名（rank method="first"，1-based）                          │
│ - 回傳 [snap_date, cust_id, prod_code, score, rank]                          │
