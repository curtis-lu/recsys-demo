  設計 - 第一部分：核心框架

  Core 組件（src/recsys_tfb/core/）：

  1. ConfigLoader - 載入 conf/{env}/ 下的 YAML 檔案，支援 base → env
  覆寫合併。提供 get_catalog()、get_parameters() 等方法。
  2. DataCatalog - 根據 catalog.yaml 定義，管理所有資料集的讀寫。每個資料集有
  type（ParquetDataset / HiveDataset）、filepath 或 table_name、partition
  等屬性。開發環境自動使用 ParquetDataset，正式環境切換為 HiveDataset。
  3. Node - 封裝一個純函數 + 其輸入/輸出名稱。例如 Node(func=stratified_sample, 
  inputs=["feature_table", "label_table", "params:sampling"], 
  outputs=["sampled_data"])
  4. Pipeline - Node 的有序集合，自動解析依賴順序。
  5. Runner - 執行 Pipeline，負責從 DataCatalog 取資料、傳入 Node、將輸出存回
  DataCatalog。處理日誌和計時。

  I/O 層（src/recsys_tfb/io/）：
  - AbstractDataset - 基底類別，定義 load() / save() / exists() 介面
  - ParquetDataset - 本地 Parquet 讀寫（支援 pandas 和 PySpark）
  - HiveDataset - Hive table 讀寫（PySpark）
  - PickleDataset - 模型檔等 Python 物件
